Profiel DVO

Situering binnen Digitale Vormgeving

Binnen de opleiding Digitale Vormgeving ontwikkel ik mezelf in vier rollen: creatieve vormgever, front-end ontwikkelaar, communicator/teamspeler en levenslang lerende. Hieronder licht ik toe hoe ik mezelf momenteel situeer binnen elk van deze rollen.

Creatieve Vormgever

Ik kan een opdracht analyseren en omzetten naar een visueel ontwerp dat aansluit bij de vraag van de opdrachtgever. Tijdens WPL1 heb ik vooral gewerkt aan mijn portfolio-website, waar ik moderne UI-technieken toepaste zoals glassmorphism, animaties en responsive grids. Ik leer steeds beter hoe ik een ontwerp omzet naar een duidelijke structuur en hoe ik keuzes visueel verantwoord.

Front-end Ontwikkelaar

Ik kan HTML, CSS en JavaScript toepassen om een ontwerp om te zetten naar een functionele website. Tijdens mijn portfolio-opdracht heb ik gewerkt met componenten, variabelen, animaties en responsive gedrag. Ik leer nog hoe ik code efficiënter kan structureren, maar ik groei snel en kan zelfstandig problemen oplossen.

Communicator & Teamspeler

Ik communiceer duidelijk, vraag feedback en kan mijn keuzes uitleggen. In groepsopdrachten werk ik constructief samen en denk ik mee over oplossingen. Ik gebruik vakjargon waar nodig en kan mijn werk presenteren op een begrijpelijke manier.

Levenslang Lerende

Ik ben zelfkritisch en probeer mezelf constant te verbeteren. Ik volg nieuwe trends in design en front-end en pas ze toe in mijn werk. Ik werk zorgvuldig, respecteer privacy en probeer steeds professioneler te worden in mijn aanpak.

AIM-Reflectie

Tijdens WPL1 heb ik veel geleerd over hoe ik een project moet aanpakken. Ik heb ontdekt dat ik sterk ben in visuele keuzes en structuur, maar dat ik soms te perfectionistisch ben waardoor ik tijd verlies. Ik heb geleerd om sneller beslissingen te nemen en mijn werk iteratief te verbeteren.

Wat neem ik mee naar WPL2?

In WPL2 wil ik efficiënter werken, beter plannen en mijn code nog netter structureren. Ik wil ook meer experimenteren met JavaScript en animaties om mijn websites interactiever te maken.

Mijn X-factor

Mijn X-factor is discipline. Dankzij karate heb ik doorzettingsvermogen, focus en rust onder druk. Dat helpt me om projecten af te maken, ook wanneer het moeilijk wordt. Ik blijf leren, verbeteren en groeien.
